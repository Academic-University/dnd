## Энциклопедия ДнД мира

Клуб настольных ролевых игр АУ проводит регулярные собрания по игре Dungeons & Dragons.
Этот репозиторий содержит описания ключевых аспектов игрового мира: географии, истории, рас, существ и правил взаимодействия.

Сайт доступен на [GitHub Pages](https://academic-university.github.io/dnd/).

## Структура контента

Материал разделён тематически по [книгам](https://github.com/Academic-University/dnd/tree/main/content), за создание которых отвечает огранизация из игрового мира, **Независимая Библиотека**. Сейчас в катологе есть следующие книги:

- **Общая география.** Информация про страны, города, географические особенности регионов.
- **Общая история.** Информация о наиболее важных исторических событиях.
- **Общая культура.** Более подробно про культуру и традиции.
- **Общее мироустройство.** Современные представления о работе мира.
- **Каталог божественных сущностей.**
